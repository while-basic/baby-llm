Examples
========

NeuralChild comes with several example scripts demonstrating various features.

.. toctree::
   :maxdepth: 2

   basic_simulation
   mother_child_interaction

All examples are located in the ``examples/`` directory of the repository.

Running Examples
----------------

.. code-block:: bash

   cd examples
   python basic_simulation.py

Available Examples
------------------

1. **Basic Simulation** - Simple simulation demonstrating core functionality
2. **Mother-Child Interaction** - Detailed interaction patterns and developmental progression

See the :doc:`../user_guide/index` for more usage patterns.
