Web Interface
=============

The system includes a Flask-based web interface for monitoring and interaction.

Starting the Web Interface
--------------------------

.. code-block:: bash

   python -m neural_child --web

Access the dashboard at: ``http://localhost:5000``

Features
--------

* Real-time dashboard
* Emotional state visualization
* Developmental metrics display
* Chat interface
* Progress tracking
* Stage progression indicators
* Alert system
